Downloading Convolution

On this page you can download Convolution and any required software.

The game can be downloaded in three main ways:

  • TADS 3 file (576 kb)
    (Preferred) If you're a frequent IF player or do not have a Windows machine, download this small t3 file. You will need the latest TADS 3 player's kit.
  • Simple executable (2.09 Mb)
    This executable runs on Windows machines. It contains the game file, as well as the game engine all wrapped into one. This is a bigger download but requires less effort.
  • Installer (1.01 Mb)
    The Windows installer is a single executable file that will install the executable version, as well as put in Start menu items. This is still a large file, but is smaller than the executable alone.

The current version of Convolution is pre-alpha. This means there is nothing available to download yet.

Hints, walkthroughs and other material can be found on the information page.

If you would like to find more interactive fiction games, I recommend you visit the IF archive.
